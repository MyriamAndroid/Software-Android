Política de Privacidad de AnaliVal

Fecha de entrada en vigor: 29 de abril de 2026

AnaliVal es una aplicación para analizar archivos CSV seleccionados por el usuario en su dispositivo.

1. Información que recopilamos

AnaliVal no recopila datos personales en servidores externos. AnaliVal no crea cuentas obligatorias ni solicita inicio de sesión para sus funciones principales.

2. Archivos y datos procesados

La aplicación accede únicamente a los archivos que el usuario selecciona de forma explícita (por ejemplo, archivos CSV). El procesamiento de estos datos se realiza localmente en el dispositivo.

3. Compartición de datos

AnaliVal no vende, comparte ni transfiere datos personales del usuario a terceros.

4. Permisos

  • Acceso a archivos/documentos: se utiliza únicamente para que el usuario pueda seleccionar y analizar su archivo CSV.

5. Seguridad

AnaliVal está diseñada para minimizar el tratamiento de datos y realizar el análisis localmente.

6. Menores de edad

AnaliVal no está dirigida específicamente a menores de 13 años.

7. Cambios a esta política

Esta Política de Privacidad puede actualizarse cuando sea necesario. La versión vigente estará siempre disponible en esta URL.
